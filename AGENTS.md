# serv00-web-app-installer

This project is a compliance-first Serv00 free-tier web app installer.

Do not add features for AList/OpenList/abcd, traffic proxy/VPN/tunneling, traffic relay, Nezha, sing-box, high-frequency keepalive, account farming, process hiding, or random binary names.

Keep functionality limited to ordinary web hosting: static/PHP sites, lightweight Node/Python web apps, Serv00 local reverse-proxy website type for those web apps, SSL, logs, and low-frequency health checks.
